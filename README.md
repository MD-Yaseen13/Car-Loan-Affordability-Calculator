This **Car Loan Affordability Calculator** is a web application built using Streamlit that helps users determine if they can afford a car loan. It calculates essential metrics such as EMI, down payment, total loan amount, and total monthly costs. The app also provides interactive visualizations and detailed affordability insights.

---

## Features
- **Interactive Input Fields**: Users can input car price, monthly income, loan terms, and more to get tailored results.
- **Affordability Analysis**: Calculates if the car loan fits within the user's budget based on a 10% income threshold.
- **Cost Breakdown**: Displays down payment, EMI, loan amount, and total cost of ownership.
- **Visualization**:
  - Pie chart of cost breakdown.
  - Line graph for EMI schedule (principal vs. interest).
  - Amortization table.
- **Indian Currency Formatting**: Numbers are formatted to display values in Indian Rupees (â‚¹).

## How to Use
1. Visit the live application: [Car Loan Affordability Calculator](https://car-loan-affordability-calculator.streamlit.app/).
2. Enter the following details:
   - Car price
   - Down payment percentage
   - Monthly income
   - Loan term (in years)
   - Interest rate
   - Expected monthly fuel cost
3. View:
   - Detailed loan affordability analysis.
   - Monthly cost breakdown.
   - Loan amortization schedule.
4. Use the visualizations to understand the breakdown of costs.
